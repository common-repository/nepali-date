Nepali Date is a plugin to display the current nepali date on your website.

== Description ==

Nepali Date is a simple widget that displays the nepali date. The plugin is relatively easy to setup.
Added shortcode to display date in posts and pages Use Shortcode [nepalidate] to display in post and pages.
== Installation ==

This plugin is installed just like any other WordPress plugin. 
More [detailed installation instructions]
(http://codex.wordpress.org/Managing_Plugins#Installing_Plugins "Installing Plugins - WordPress Codex") are available on the WordPress Codex.

0. Use Shortcode [nepalidate] to display in post and pages.
1. Download Nepali Date
2. Extract nepalidate.php
3. Upload this file to to `../wp-contents/plugins/`
4. Activate the plugin in *Dashboard > Plugins*
5. Drag and drop drom widget area.
7. Enjoy. And then consider donating

== Use ==

After activating this plugin, simply visit the site and it should be visible in your sidebar.
